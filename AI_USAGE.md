# AI Usage Declaration

AI tools were used as a learning and productivity aid during the development of this project.

## Areas of AI Assistance

### Project Planning

AI was used to discuss project structure, folder organization, and feature planning.

### Technical Guidance

AI provided assistance in:

* React component organization
* Express API development
* MongoDB schema design
* Error debugging
* Deployment guidance

### Cloudinary Integration

Cloudinary was a completely new concept for me before working on this project.

I used AI assistance to understand:

* Why cloud storage is preferred over local storage
* How Cloudinary stores files globally
* How secure URLs are generated
* How uploaded files can be accessed from anywhere
* How Cloudinary integrates with Express and Multer

This helped me understand modern approaches to file management in web applications.

### Documentation Support

AI was used to improve documentation structure and technical explanations.

## Developer Contribution

All project setup, implementation, integration, testing, debugging, and final submission preparation were performed by me.

AI was used as a learning assistant to understand concepts and resolve issues. All code was reviewed, modified, tested, and integrated manually.

## Reflection

The most valuable learning from this project was understanding cloud-based file storage through Cloudinary and how uploaded files can be managed efficiently in modern full-stack applications.

The project also improved my understanding of React, Node.js, Express, MongoDB, API development, and system integration.
